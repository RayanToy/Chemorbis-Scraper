\# ChemOrbis Price Data Scraper



Автоматизированный веб-скрапер для сбора данных о ценах на полимеры с платформы \[ChemOrbis.com](https://www.chemorbis.com). Консолидирует еженедельные ценовые индексы и данные графиков в структурированный Excel-файл для дальнейшего анализа.



\## Описание



Проект автоматизирует ручной процесс сбора данных о ценах на полимерное сырьё с ChemOrbis — ведущей платформы для мониторинга нефтехимического рынка. Ранее этот процесс занимал несколько часов в неделю, теперь выполняется автоматически за \~30 минут.



\### Что делает скрапер:



\- \*\*Авторизация\*\* на платформе с обработкой всплывающих окон и shadow DOM

\- \*\*Сбор данных Price Index\*\* — загрузка еженедельных Excel-отчётов с ценами Low/Avg/High

\- \*\*Сбор данных Price Wizard\*\* — экспорт CSV-данных временных рядов из интерактивных графиков

\- \*\*Консолидация данных\*\* — объединение, очистка и стандартизация данных из обоих источников

\- \*\*Форматирование Excel\*\* — создание готового к анализу файла с корректными типами данных

\## Возможности



\- \*\*Механизм повторных попыток\*\* — настраиваемое количество ретраев с таймаутом для каждой задачи

\- \*\*Модульная архитектура\*\* — отдельные модули для авторизации, скрапинга, обработки, форматирования

\- \*\*Гибкая настройка\*\* — все URL, таймауты и маппинги данных вынесены в `config.yaml`

\- \*\*Безопасность\*\* — логин/пароль хранятся в `.env` и не попадают в Git

\- \*\*Логирование\*\* — структурированные логи в консоль и файл

\- \*\*CLI-интерфейс\*\* — запуск только скрапинга, только обработки или полного пайплайна

\## Установка



```bash

git clone https://github.com/rayantoy/chemorbis-scraper.git

cd chemorbis-scraper



python -m venv venv

source venv/bin/activate      # Linux/Mac

venv\\Scripts\\activate         # Windows



pip install -r requirements.txt

```



\### Необходимо



\- \*\*Google Chrome\*\* установлен

\- \*\*ChromeDriver\*\* совместимый с версией Chrome

\- \*\*Аккаунт ChemOrbis\*\* с доступом к ценовым данным



\### Настройка



Создай файл `.env` в корне проекта:



```env

CHEMORBIS\_LOGIN=ваш\_логин

CHEMORBIS\_PASSWORD=ваш\_пароль

DOWNLOADS\_DIR=C:/Users/youruser/Downloads

BASE\_OUTPUT\_DIR=C:/Users/youruser/ChemOrbis\_Data

```



\## Использование



```bash

\# Полный пайплайн: скрапинг + обработка

python main.py



\# Только скрапинг (без обработки)

python main.py --scrape-only



\# Только обработка ранее скачанных файлов

python main.py --process-only



\# Свой конфиг

python main.py --config my\_config.yaml

```



\## Конфигурация



\### `config.yaml`

\- \*\*URLs\*\* — страницы авторизации, Price Index, Price Wizard

\- \*\*Параметры скрапинга\*\* — таймауты, количество ретраев, задержки

\- \*\*Маппинги данных\*\* — замены названий котировок, валюты, исключения



\### Входной Excel-файл

Файл с выборкой котировок размещается в `data/quotes\_selection.xlsx` с листами:

\- \*\*PriceIndex\*\* — колонки: Territory, Product Group, Product, Transaction Type

\- \*\*PriceWizard\*\* — колонки: Territory, Product Group



\## Выходные данные



Результат сохраняется в `output/chemorbis\_consolidated.xlsx`:



| Колонка | Описание |

|---|---|

| Quote | Название ценового индекса / котировки |

| Currency | Валюта (USD / EUR) |

| UOM | Единица измерения (mt — метрическая тонна) |

| Level | Уровень цены (Low / Avg / High) |

| Price | Значение цены |

| Date | Дата публикации |

| Agency | Источник данных (Chemorbis) |

| Month | Номер месяца |

| Year | Год |

| Week | Номер недели ISO |

| Week Start Date | Начало ценовой недели |

\## Контекст проекта



Инструмент разработан для автоматизации еженедельного сбора данных мониторинга цен на полимерное сырьё в нефтехимической отрасли. Автоматизация позволила сократить время сбора данных с нескольких часов до \~30 минут без участия оператора.



\## Дисклеймер



Проект представлен в образовательных целях и как элемент портфолио. Веб-скрапинг должен соответствовать условиям использования целевого сайта. Для доступа к данным ChemOrbis необходима действующая подписка.
